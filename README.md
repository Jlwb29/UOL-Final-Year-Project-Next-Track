UOL Final Year Project (Next Track)

This project follows Template 7.2: Project Idea 2 – Next Track: A Music Recommendation API

This project presents Next Track, a stateless RESTful music recommendation API
that generates recommendations using metadata-based feature vectors stored in
MySQL. 

Similarity based ranking is applied to produce recommendation. 

The system is evaluated using cold start and sparse input test cases and the results
indicate that metadata only approaches can produce coherent recommendations
proving its suitability for privacy aware and lightweight music recommendation
applications.
